## 📚 Sobre

Este repositório reúne projetos e códigos de estudo voltados para **Sistemas Embarcados**, explorando diferentes plataformas de microcontroladores.

O objetivo é desenvolver habilidades práticas em programação de baixo nível, controle de hardware e integração entre software e eletrônica.

---

## 🧠 Plataformas Utilizadas

* 🔹 Arduino
* 🔹 ESP32
* 🔹 Tiva-C

Cada plataforma possui sua própria pasta com códigos e experimentos específicos.

## 🎯 Objetivo

Consolidar conhecimentos em **programação embarcada**, preparando para projetos mais complexos envolvendo automação, IoT e sistemas embarcados profissionais.
